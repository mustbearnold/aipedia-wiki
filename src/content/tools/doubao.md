# Doubao

ByteDance's consumer AI assistant, launched August 2023 and the most-used AI product in China by daily active users. Runs **Doubao Seed 2.0** as of February 14, 2026, with text, image (Seedream), video (Seedance), voice, and document analysis in one app. Seed 2.0 remains the current flagship through the May 2026 refresh.

No direct US app store presence. International users access similar capability through the Cici / Dola app in supported markets. API access runs through ByteDance's Volcano Engine cloud.

## What it actually is

A single consumer product covering chat, image gen, video gen, voice, and document analysis. ByteDance positions it against GPT-5.2 and Gemini 3 Pro on benchmarks, against DeepSeek on Chinese-market share, and against TikTok's own distribution engine for user growth.

The underlying Seed 2.0 family ships as an API on Volcano Engine. Seed 2.0 Pro handles reasoning, Seed 2.0 Code handles programming tasks, Seed 2.0 Lite and Mini handle high-volume cheap workloads. Pricing sits among the lowest for any frontier-tier model, Chinese or Western.

The real moats are distribution and cost. Douyin integration funnels users at a rate Western assistants cannot match inside China. Volcano Engine's pricing reflects ByteDance's willingness to run margin-negative to capture API market share.

## When to pick Doubao

- **Chinese-language consumer use.** Free, polished, integrated with Douyin, and optimized for Mandarin idiom and task flow.
- **High-volume API workloads in Asia.** Volcano Engine pricing crushes Western equivalents on bulk token spend when data residency on Chinese infrastructure is acceptable.
- **Cici / Dola users abroad.** The international build supports 18 languages and runs on the same underlying model family.
- **Multimodal tasks bundled in one app.** Seedream for image, Seedance for video, voice chat, all inside the consumer product at zero cost.
- **Research benchmarking against GPT-5.2 or Gemini 3 Pro.** Seed 2.0 Pro sits close enough to warrant head-to-head testing on your own prompts.

## When to pick something else

- **Primary English chat:** [ChatGPT](/tools/chatgpt/) or [Claude](/tools/claude/). Doubao's English UX is a translation layer, not a first-class surface.
- **Western data residency:** [ChatGPT](/tools/chatgpt/), [Claude](/tools/claude/), or [Gemini](/tools/gemini/). Volcano Engine processes tokens on Chinese infrastructure.
- **Cheapest API with easier Western signup:** [DeepSeek](/tools/deepseek/). Similar pricing posture, lower regulatory exposure for most Western developers.
- **Open weights for self-hosting:** [Qwen](/tools/qwen/) or [GLM](/tools/glm/). Doubao weights are not public.
- **Google Workspace integration:** [Gemini](/tools/gemini/). Doubao has no Workspace hooks.

## Failure modes

- **Volcano Engine signup is Chinese-only in practice.** Real-name verification with a Chinese phone number is the default path. Western developers face meaningful friction before the first API call.
- **No direct US app store presence.** "Doubao" does not ship in the US App Store or Google Play. International users go through Cici or Dola, which lags the Chinese product on features.
- **Benchmark claims are vendor-published.** ByteDance's GPT-5.2 and Gemini 3 Pro parity numbers have not been independently verified at the depth Claude or GPT results have. Test on your own prompts before committing.
- **Geopolitical exposure.** ByteDance is the TikTok parent. Enterprise buyers in the US, UK, and EU face the same regulatory review that has shadowed TikTok for four years.
- **English output lags Chinese output materially.** The model handles English functionally but loses nuance, idiom, and technical precision against OpenAI frontier models or Claude Opus 4.7.
- **Cici / Dola feature lag.** The international app reaches the latest model months after the Chinese Doubao app. Users abroad rarely run the real flagship.
- **Data residency is on Chinese infrastructure.** Volcano Engine processes tokens in China. Healthcare, legal, and financial workloads subject to Western residency rules cannot use it.

## FAQ

**Is Doubao available outside China?**
Not as "Doubao." ByteDance distributes similar functionality internationally through the Cici app, rebranded Dola in some markets. Cici / Dola supports 18 languages. The Volcano Engine API is accessible globally, though registration requires a Chinese phone number.

**How cheap is the API compared to ChatGPT?**
Doubao Seed 2.0 Pro runs ~$0.47/M input and ~$2.37/M output. That lands roughly 10x below OpenAI's comparable tiers on similar volumes. Savings compound on batch workloads.

**What is Doubao Seed 2.0?**
The model family released February 14, 2026. Four variants: Pro, Lite, Mini, and Code. Pro targets GPT-5.2 and Gemini 3 Pro on math, coding, and reasoning benchmarks. Powers the Doubao app and the Volcano Engine API.

**Should Western developers use Volcano Engine?**
Only if data residency on Chinese infrastructure is acceptable and a Chinese phone number is available. For most Western teams, [DeepSeek](/tools/deepseek/) or [Qwen](/tools/qwen/) on neutral providers like OpenRouter are lower-friction paths to similar pricing.

**Does Doubao generate images and video?**
Yes, bundled in the consumer app. Images via Seedream, video via Seedance. No subscription required in the Chinese app. Cici / Dola exposes the same stack with feature lag.
