# Retell AI

Retell AI is a platform for building AI voice agents and AI chat agents. It is aimed at teams that want to automate customer calls, sales qualification, appointment workflows, support routing, and other voice-first business interactions.

The official pricing page positions Retell as "true pay as you go": start at $0, get free credits, and pay only for usage before moving to custom enterprise terms.

## System Verdict

> **Pick Retell AI when the workflow is a real phone call, not just a chatbot with voice.** Retell is strongest when you need call analytics, transcripts, simulation testing, webhooks, API access, concurrency, and enterprise escalation.
>
> **Skip it if you want simple fixed-price automation.** Voice agents are usage-based systems, and real cost depends on minutes, model choice, speech stack, telephony, and call complexity.
>
> **Who pays which tier:** Pay-as-you-go is right for builders and pilots. Enterprise is for larger deployments that need SSO, custom agreements, dedicated infrastructure, and higher concurrency.

## What It Actually Is

Retell is voice-agent infrastructure, not a generic transcription app. The pricing page lists pre-built agent templates, call analytics, transcripts, simulation testing, webhooks, API access, and 20 included concurrent calls on pay-as-you-go.

That combination is important. Good voice agents need more than a nice voice. They need interruption handling, fallback behavior, validation, call transfer logic, latency management, transcripts, testing, and post-call data flows. Retell is built around that operational layer.

## When To Pick Retell AI

- **You need phone call automation.** Retell is made for voice agents that talk with customers, not just internal dictation.
- **You need developer control.** Webhooks and API access are key for production workflows.
- **You want to test before committing.** Pay-as-you-go plus free credits lowers the first deployment barrier.
- **You need observability.** Analytics and transcripts are essential for debugging voice agents.
- **You need enterprise escalation later.** Dedicated infrastructure, SSO, custom contracts, and higher concurrency are available.

## When To Pick Something Else

- **Voice dictation:** [Wispr Flow](/tools/wispr-flow/) or [Whisper](/tools/whisper/).

## Failure Modes

- **Cost forecasting is hard.** Minutes, models, TTS, STT, telephony, transfers, and retries all affect real cost.
- **Voice agents need narrow workflow design.** Broad prompts can fail in live calls.
- **Compliance is not automatic.** Healthcare, legal, finance, and HR use cases need careful review.
- **Human fallback still matters.** Agents should transfer or escalate when confidence drops.
- **Latency and interruption handling can make or break the experience.** Test with real accents, noisy rooms, and impatient callers.
