# v0 by Vercel

Vercel's AI UI generator at [v0.app](https://v0.app). Prompts produce React and Next.js components with shadcn/ui and Tailwind, a live preview, and one-click deploy to Vercel. Blocks now render Svelte, Vue, HTML, Markdown, Bootstrap, and Material UI inside the same workspace.

Token-metered credits replaced the old message-count system. The free tier runs on $5 of monthly credits.

## What it actually is

A hosted AI workspace that outputs production-grade React. You describe a UI, v0 renders it live, and you iterate in chat or in Design Mode. The same generation produces editable code you can copy, push to GitHub, or deploy straight to Vercel.

Blocks extend the output surface. A single project can render Svelte, Vue, Bootstrap, or Material UI alongside the default shadcn/ui path. The chat sidebar was redesigned in 2026 with clearer step-by-step explanations of each generation.

The moat is the Vercel integration. One click publishes a live URL with analytics, no build config required. Figma import and GitHub sync close the design-to-production loop for shops already standardized on that stack. The weakness is scope. v0 is a component factory, not an app builder.

## When to pick v0

- **Your team ships on Next.js and shadcn/ui.** Output matches the conventions you already use.
- **You need a component in under ten minutes.** Forms, dashboards, pricing tables, landing sections all land on the first or second prompt.
- **You import from Figma or screenshots.** Both workflows produce matching shadcn/ui code without hand translation.
- **You want private generations with shared team credits.** Team at $30/user/mo bundles centralized billing and API access.
- **You deploy to Vercel.** One-click deploy with zero config is the tightest loop on the market.

## When to pick something else

- **Full-stack app with backend and auth out of the box:** [Lovable](/tools/lovable/) provisions Supabase, database, and deploy in one flow.
- **In-browser full-stack prototyping with a running Node server:** [Bolt.new](/tools/bolt/) runs WebContainers in the browser.
- **Autonomous agent that owns the whole app:** [Replit Agent](/tools/replit-agent/) builds and deploys end to end.

## Failure modes

- **Credit burn on complex prompts.** A dashboard with charts can eat a meaningful share of Free's $5 credit allowance. Budget accordingly.
- **React-ecosystem bias.** Blocks render Svelte and Vue, but the generation quality is still strongest on React plus shadcn/ui. Other stacks feel bolted on.
- **Components, not apps.** State management, routing, backend, and auth stay your problem. Pick Lovable or Bolt when scope crosses that line.
- **Generic defaults.** Output looks like every other shadcn/ui site unless you push the prompt toward brand specifics or hand-edit after export.
- **Model is not user-selectable on Free.** Vercel routes across frontier models but does not expose model choice on the base plan.
- **API is paywalled.** Premium and above. Free tier users cannot script generations.

## FAQ

**Is v0 free?**
Yes. The Free tier includes $5 of monthly credits, Design Mode, GitHub sync, and Vercel deploy. Premium at $20/mo raises credits and unlocks Figma import and the v0 API ([v0.app/pricing](https://v0.app/pricing)).

**What stacks can v0 output?**
Default output is React and Next.js with shadcn/ui and Tailwind. Blocks render Svelte, Vue, HTML, Markdown, Bootstrap, and Material UI ([v0.app/docs](https://v0.app/docs)).

**Can v0 build a full app with a backend?**
Not directly. v0 generates components and pages. For full-stack apps with backend and auth, [Lovable](/tools/lovable/) or [Bolt.new](/tools/bolt/) are better picks.

**How do credits convert?**
v0 uses token-metered billing. Input and output tokens both consume credits, so complex prompts with large context burn faster than simple ones.

**Does v0 have an API?**
Yes, on Premium and above. The free tier does not expose API access.
