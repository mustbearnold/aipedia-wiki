# Relume

Relume is an AI site-building tool built on top of Webflow, Figma, and React rather than replacing them. A text prompt generates a **sitemap**, then **wireframes** with real, editable components, then a **style guide** (colors, typography, spacing tokens). From there the project exports into Webflow, Figma, React code, or, as of May 2026, directly into **Claude Design**.

The company is bootstrapped, reports roughly **$3.3M ARR**, and says it has **1 million-plus designers and developers** and **500,000-plus websites** built on the platform. Relume runs a large **Webflow and React component library** (1,000+ components) as the backbone of every generated page.

## System Verdict

> **Pick Relume if you need a fast, structured first draft of a marketing site inside Webflow or Figma.** The sitemap-then-wireframe-then-style-guide flow compresses a day of scaffolding into minutes, and the component library keeps output editable rather than locked into a black-box template.
>
> **Skip it if you want one tool that designs and hosts the finished site.** Relume is a design-acceleration layer. Webflow exports still need a Webflow plan to publish, Figma exports need a Figma seat, and React exports still need a developer to wire them up. Buyers expecting an all-in-one builder like Framer AI or Lovable should look there instead.
>
> **Who pays which tier:** Free for testing the workflow on one project and one wireframe page. Starter (reported around $26/mo) for solo freelancers who need unlimited AI usage on a small handful of projects. Pro (reported around $40-58/mo, sources disagree) for anyone running unlimited projects. Team (reported around $36-52/mo per seat, 3-seat minimum) for agencies collaborating in a shared workspace. Confirm the exact figure at checkout since Relume's public pricing page renders dynamically and third-party trackers do not agree.

## What it actually is

Relume is not a standalone website host. It is a generation and component layer that sits in front of Webflow, Figma, and now Claude Design. The **AI Site Builder** takes a prompt, produces a full sitemap, then turns each sitemap entry into a wireframe page built from real, swappable components rather than static images. A **Style Guide Builder** generates color tokens, typography scales, and spacing rules that keep every page visually consistent.

The **component libraries** (Webflow, React, Figma) are the product's actual moat. Relume ships 1,000+ Webflow and React components and 1,500+ Figma components, all built to snap into the generated wireframes without a redesign pass.

**Claude Design Export**, launched May 6, 2026, exports a full Relume design system (colors, typography, spacing, components) into Claude Design in one click. Relume's pitch is that Claude Design alone tends to produce generic, drifting output past the first page, while feeding it a Relume-generated design system keeps every subsequent page on-brand.

**Relume Library MCP**, launched July 1, 2026, exposes the component library to AI coding assistants (Cursor, Claude, Windsurf, VS Code) via the Model Context Protocol, so an agent can pull a real, editable, on-brand component by description instead of hallucinating markup from scratch.

## When to pick Relume

- **You need a Webflow sitemap and wireframe fast.** The prompt-to-sitemap-to-wireframe flow is the fastest verified path to a structured first draft inside Webflow.
- **You are pairing with Claude Design and want brand consistency.** Claude Design Export solves the generic-output and page-drift problem Claude Design has on its own.
- **You are an agency running multiple client sites off one component system.** The shared Figma/Webflow/React libraries keep output consistent across projects.
- **You use an AI coding assistant and want real components, not guesses.** The Relume Library MCP pulls actual, editable components with design tokens attached into Cursor, Claude, Windsurf, or VS Code.
- **You want an editable wireframe, not a locked template.** Every generated page is built from swappable components, not a flattened image.

## When to pick something else

- **All-in-one AI site builder that also hosts the result:** [Framer AI](/tools/framer-ai/) or [Lovable](/tools/lovable/). Both design and publish without a second downstream tool.
- **Instant web-app prototyping with real code output:** [v0](/tools/v0/) or [Bolt](/tools/bolt/). Better when the buyer wants functioning React/Next.js, not a marketing-site wireframe.
- **Mature, pixel-accurate design system work:** [Figma](/tools/figma/). Deeper design-token and component tooling once the project moves past first draft.
- **Fast editable mockups for non-designers:** [Uizard](/tools/uizard/). Lighter-weight and cheaper at entry for solo PMs and founders.
- **The Claude Design canvas itself:** [Claude Design](/tools/claude-design/). Relume is a design-system feeder for it, not a replacement.
- **AI-native design exploration without Webflow lock-in:** [Google Stitch](/tools/google-stitch/).

## Recent developments

Relume shipped two notable releases in mid-2026. **Claude Design Export** (May 6, 2026) lets users generate a full design system in Relume and push it into Claude Design in one click, aimed at fixing the generic, drifting output Claude Design produces without a defined design system. Relume frames this as step one of a deeper Claude partnership.

The **Relume Library MCP** (July 1, 2026) is the more structurally significant move: it exposes Relume's 1,000+-component library through the Model Context Protocol so AI coding assistants (Cursor, Claude, Windsurf, VS Code) can pull real, on-brand, editable components by description instead of generating markup from scratch. This puts Relume's component library directly inside the AI-coding workflow rather than only the Webflow/Figma design workflow.

Earlier 2025-2026 updates included Wireframing 2.0 and Claude-powered copywriting (September 2025), a saved-components and page-templates feature (August 2025), and a Design View launch with smarter style-guide generation (July 2025), showing a steady cadence of monthly-to-quarterly feature releases rather than a single big-bang launch.

## System verdict

Relume earns its keep as the fastest verified path from a text prompt to an editable Webflow or Figma first draft, and the May and July 2026 Claude integrations extend that value into the AI-coding and AI-design-canvas workflows rather than keeping it siloed in Webflow. The catch is that Relume was never meant to be the last tool in the chain: every export still needs a Webflow plan, a Figma seat, or a developer to finish the job, and the pricing page's client-side rendering combined with disagreeing third-party trackers make it harder than it should be to know the exact bill before signing up. Buyers should treat the Starter price as reliable and confirm Pro and Team figures directly at checkout.

## Failure modes

- **Pricing page renders prices client-side.** Automated and even some manual checks of relume.io/pricing return plan names and limits but not dollar figures without a full browser render, which is unusual for a SaaS pricing page and makes public price transparency worse than competitors.
- **Third-party price trackers disagree.** Independent sources converge on Starter around $26/mo but split on Pro ($40 vs $58/mo) and Team ($36 vs $52/mo per seat). Confirm live pricing before budgeting.
- **Not a standalone publisher.** Every export path (Webflow, Figma, React) requires a separate downstream tool and, for Webflow, a separate paid plan to actually go live.
- **Starter's project cap is a lifetime total, not monthly.** Community documentation confirms exceeding the cap requires deleting an existing project rather than waiting for a monthly reset.
- **Component-first output still needs a design pass.** Generated pages are structurally sound but templated; teams with strong brand guidelines will want to customize before shipping.
- **Roadmap uncertainty on pricing model.** Relume has teased an AI-usage-based Publish feature that could restructure pricing away from flat per-seat plans; current plan holders should watch for changes.
