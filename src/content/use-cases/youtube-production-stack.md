## Tool Details

### Finding Content (Daily Habit)
- **Product Hunt** (producthunt.com): check daily launches. 300+ upvotes on launch day = video-worthy.
- **There's An AI For That** (theresanaiforthat.com): comprehensive new tool tracker
- **Futurepedia** (futurepedia.io): curated, newsletter available
- **Twitter/X:** #aitools, follow AI VCs. Announcements hit X before YouTube.
- **This wiki:** every tool page has a "Video Potential" section

### Voice Selection
Use ElevenLabs' "calm, authoritative male/female narrator" preset. Similar to AICodeKing's style:
- Clear, slightly warm tone
- Moderate pace (not rushed)
- Avoid overly energetic. AI tools audience prefers informative over hype.

### Editing Workflow
1. Import screen recording + voiceover into CapCut
2. Sync audio to key screen moments
3. Add zoom effects on important UI elements (CapCut's keyframe zoom)
4. Add text callouts for key features/prices
5. Cut dead air. Keep it tight.
6. Export 1080p, 30fps

### Thumbnail Formula (for this niche)
- Tool logo/screenshot on left
- Bold text (3-5 words max) on right
- High contrast background (dark preferred)
- Optional: face/avatar in corner (+20-30% CTR)
- Test: with and without face, track CTR in YouTube Analytics
