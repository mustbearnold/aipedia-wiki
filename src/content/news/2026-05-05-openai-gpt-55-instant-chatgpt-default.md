# OpenAI makes GPT-5.5 Instant the default ChatGPT model

OpenAI announced on May 5, 2026, that GPT-5.5 Instant is replacing GPT-5.3 Instant as the default ChatGPT model for all users. The model is also rolling into the API as `chat-latest`, making this a consumer and developer-platform update rather than a paid-tier-only flagship release.

The headline claims are practical: OpenAI says GPT-5.5 Instant produces fewer hallucinated claims on high-stakes prompts, gives tighter responses with less overformatting, improves image and STEM tasks, and better decides when to use web search. OpenAI also says it uses fewer words and lines in everyday writing tasks, which matters because Instant handles the highest-volume ChatGPT usage.

The update also adds memory sources across ChatGPT models. When a response uses personalization, users can see selected context sources, such as saved memories or past chats, and then delete or correct outdated items. OpenAI says shared chats do not expose those memory-source details to other viewers.

## Why this matters

Default-model changes are bigger than premium-model launches for most buyers. Free, Go, Plus, Business, and Enterprise users feel the default assistant first, and most casual users never open a model picker.

The memory-source control is also a trust feature. As ChatGPT leans harder on past chats, files, and connected data, users need visibility into why an answer sounds personalized and a way to correct stale context.

## Buyer take

For ChatGPT users, the main evaluation point is no longer whether GPT-5.5 exists. It is whether GPT-5.5 Instant improves everyday reliability enough that Plus or Pro upgrades are reserved for heavier reasoning, Codex, and agent workflows.

Enterprise teams should recheck internal guidance that still names GPT-5.3 Instant as the default. Training materials, policy docs, and model-access tables are now stale.

## What is still unclear

Availability details vary by plan, region, and surface. Enhanced personalization from past chats, files, and connected Gmail begins with Plus and Pro on web, while broader plan and mobile rollout is staged.
