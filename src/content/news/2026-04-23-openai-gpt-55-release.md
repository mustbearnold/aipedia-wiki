**OpenAI released GPT-5.5 on April 23, 2026**, making it the new top-line OpenAI model for complex work across ChatGPT and Codex.

## What changed

OpenAI describes GPT-5.5 as a model built for real computer work: coding, research, data analysis, documents, spreadsheets, software operation, and tool use across multi-step tasks.

Key claims from the launch:

- Better agentic coding and long-horizon task completion than GPT-5.4.
- Same real-world per-token latency as GPT-5.4 despite higher capability.
- Fewer tokens used on Codex tasks, improving effective cost for long-running work.
- Stronger computer-use, knowledge-work, cyber, and early scientific-research performance.

## Availability

- **ChatGPT:** GPT-5.5 Thinking is rolling out to Plus, Pro, Business, and Enterprise.
- **ChatGPT Pro:** GPT-5.5 Pro is rolling out to Pro, Business, and Enterprise.
- **Codex:** GPT-5.5 is available to Plus, Pro, Business, Enterprise, Edu, and Go plans with a 400K context window.
- **API:** announced as coming soon.

## API pricing announced

OpenAI said API pricing will be:

| Model | Input | Output | Context |
|---|---:|---:|---:|
| gpt-5.5 | $5 / 1M tokens | $30 / 1M tokens | 1M |
| gpt-5.5-pro | $30 / 1M tokens | $180 / 1M tokens | not fully specified |

Batch and Flex are listed at half the standard API rate. Priority processing is 2.5x standard.

## Why it matters for the tool wiki

This is a major scoring event for **ChatGPT** and **Codex**:

- ChatGPT gets a higher ceiling for professional research, data, and document work.
- Codex becomes more credible as a long-running engineering agent, not just a coding assistant.
- API pricing is materially higher than cheaper frontier and open models, so the value question becomes token efficiency plus task completion quality.
