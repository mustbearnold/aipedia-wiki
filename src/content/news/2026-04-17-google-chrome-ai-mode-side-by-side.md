Google launched **AI Mode in Chrome desktop** on April 17, 2026. The update makes Chrome the first mainstream browser where AI search and traditional browsing live **side-by-side** instead of swapping out.

## What changed

Previously, clicking a link from an AI-generated answer replaced the AI view with the webpage. Users had to hit Back to get the AI context again. The new design:

**1. Side-by-side view**

When you search in AI Mode and click a link, Chrome opens the site in a split pane next to the AI assistant. AI context stays visible. You can ask follow-up questions while reading the webpage.

**2. Tab context pull-in**

A new plus menu (desktop and mobile) lets you reference content from other open tabs inside an AI Mode query. Works for images, PDFs, and text. Example: drop a PDF into an AI Mode question and ask about it without switching tabs.

**3. Tab-hopping addressed**

Google specifically calls out **tab-hopping** (the context break from flicking between search results and the pages they surface) as the friction this release targets. The side-by-side model preserves the AI reasoning thread while you explore sources.

## Availability

- **Live:** US, Chrome desktop first
- **Coming soon:** Wider geographic rollout
- **Mobile:** Tab-pull integration ships alongside; side-by-side is desktop-primary

## Why this matters

**For Google:** Chrome has 65%+ desktop browser share. Shipping AI Mode as a native browser feature, not a separate product, makes **Gemini the default AI assistant for the internet** at massive scale. No download, no signup, no switch cost.

**For Perplexity Comet:**

[Comet](/tools/perplexity/) was the first serious AI-native browser with side-by-side browsing and agent capabilities. Chrome just shipped the core feature at billion-user distribution. Perplexity needs to differentiate on agent capability and specialized research workflows, not on the browsing-plus-AI UX.

**For ChatGPT Atlas:**

OpenAI's Atlas browser ships the same month. Chrome's AI Mode launch is Google's direct response, leveraging existing Chrome distribution as the moat. Atlas has to earn installs; AI Mode arrives by Chrome auto-update.

**For Arc Browser:**

Arc's "Browse for me" agent workflow was the concept Chrome and Atlas are now industrializing. Arc needs to move faster on its agent roadmap or lose the lead.

## How the browsers compare now

| Browser | AI model | Side-by-side | Agent | Distribution |
|---|---|---|---|---|
| **Chrome + AI Mode** | Gemini 3.1 Pro | Yes (April 17) | Partial (Search Labs) | Massive, auto-update |
| **ChatGPT Atlas** | GPT-5.4 | Yes | Yes | Growing, opt-in install |
| **Perplexity Comet** | Perplexity + frontier models | Yes | Yes, Comet Assistant | Comet Max ($20/mo) or free tier |
| **Arc** | Multiple | Yes | Browse for me | Smaller install base |
| **Safari** | Apple Intelligence + ChatGPT | Partial | No | Default on Apple |

## What this means for Gemini users

Short-term: AI Mode usage will climb sharply as Chrome users discover the side-by-side view in their normal workflow.

Medium-term: Gemini gets **the richest contextual signal of any AI assistant** because Chrome sees what users browse. That feedback loop is hard to replicate unless you also own a browser.

For [Gemini](/tools/gemini/), this is the distribution unlock that matters more than any model release.
