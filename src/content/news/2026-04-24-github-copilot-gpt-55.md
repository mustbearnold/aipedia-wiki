GitHub made **GPT-5.5 generally available in GitHub Copilot** on April 24, 2026.

## What changed

GPT-5.5 is now available inside Copilot for:

- Copilot Pro+
- Copilot Business
- Copilot Enterprise

GitHub lists the premium-request multiplier at **7.5x**. That puts GPT-5.5 in the same high-cost usage lane as other top frontier models inside Copilot's model picker.

## Why it matters

This closes the gap between OpenAI's GPT-5.5 launch and the tools developers actually use in the editor. Copilot now has both Anthropic's Opus 4.7 and OpenAI's GPT-5.5 in the same paid model picker.

For Copilot users, the practical question is no longer whether the model is available. It is whether a task justifies the premium-request burn. Routine autocomplete and small edits should stay on cheaper models; hard refactors, architecture questions, and agentic coding loops are the better GPT-5.5 use cases.

For the wiki score, this strengthens Copilot's utility and moat without changing the value risk: high-end models are available, but the premium request pool is finite.
