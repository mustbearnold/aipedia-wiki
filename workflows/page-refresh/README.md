# Page Refresh Workflow

Placeholder for non-tool page refreshes and top-layer maintenance.

Use this lane for homepage, category, comparison, guide, answer, trend, workflow, and archive pages that are not covered by the tool-refresh or news-refresh workflows.

Mandatory principles:

- verify volatile facts against current sources
- update parent/top-layer pages when child content changes
- regenerate `PAGE_REFRESH_LEDGER.md`
- run verification scoped to rendered/SEO/conversion risk
- route QA mobile widths when rendered layout changes
