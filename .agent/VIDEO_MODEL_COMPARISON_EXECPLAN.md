# Video Model Comparison ExecPlan

## 1. Objective

Rebuild the stale AI video comparison at `/compare/kling-vs-seedance-vs-runway/` into a current, mobile-first buyer decision page for Seedance 2.0, Kling 3.0, Google Veo 3.1, and Runway. The goal is to improve trust, search relevance, internal linking, and revenue routing for one of AiPedia's most commercially important categories.

## 2. Current state

The existing comparison file is `src/content/comparisons/kling-vs-seedance-vs-runway.md`. It compared only Kling, Seedance, and Runway, used stale April 2026 metadata, made unsupported pricing claims, omitted Veo 3.1, and framed Seedance through the wrong source route. The comparison route is generated by `src/pages/compare/[slug].astro` and rendered by `src/layouts/ComparisonLayout.astro`, with CTA cards handled through `src/components/CommercialCTA.astro`.

## 3. Target state

The page keeps the existing slug for URL continuity but becomes a four-way split-decision buyer page. Mobile readers should see an immediate answer: Seedance for raw model testing, Kling for creator/cinematic value, Veo for Google/API/provenance, and Runway for production workflow. Desktop readers can still use the canonical fact table and richer comparison layout.

## 4. Scope

Included: comparison frontmatter, comparison body, current official source links, internal links, and any small related-link fixes needed to avoid stale anchor text. Excluded: full category rewrite, controlled benchmark execution, new analytics schema, and broad Runway tool-page rewrite.

## 5. Files likely affected

- `src/content/comparisons/kling-vs-seedance-vs-runway.md`
- `src/content/tools/runway.md`
- `src/content/use-cases/best-ai-video-generator.md`
- `.agent/VIDEO_MODEL_COMPARISON_EXECPLAN.md`

## 6. Data model impact

The comparison `tools` list expands from `[kling, seedance, runway]` to `[seedance, kling, veo, runway]`. `winner` changes from `kling` to `depends` so the template does not force a false single winner. No schema migration is required.

## 7. SEO impact

The title, meta description, `last_verified`, and body now target the real search intent: Runway vs Kling vs Seedance vs Veo, best AI video model, best video generator, and buyer-route comparison. The old slug is retained to preserve existing route value.

## 8. Conversion impact

The rebuilt page routes users by job-to-be-done instead of pushing one generic winner. Existing comparison CTA cards keep tracked `CommercialCTA` placement metadata for each tool. The content gives Runway a clear production-workspace buyer route without overstating raw model quality.

## 9. Mobile UX impact

The primary content uses short sections and bullet decisions instead of a wide markdown table as the main mobile experience. The template's comparison cards and fact table provide richer layout while the text remains readable at narrow widths.

## 10. Implementation steps

1. Verify current official sources for Seedance, Kling, Veo, and Runway.
2. Replace stale comparison frontmatter and content.
3. Update stale related/internal anchors if the old three-way label remains.
4. Run provenance/build/smoke checks.
5. Open the local page in the in-app browser and inspect mobile/desktop basics.

## 11. Verification commands

- `npm run audit:provenance`
- `npm run build:fast`
- `npm run smoke:visual`

## 12. Acceptance criteria

The comparison page builds, includes all four current tools, avoids fake pricing and fake benchmark claims, links to official sources, preserves the old URL, gives clear buyer decisions, and does not introduce mobile-wide markdown tables as the primary comparison UI.

## 13. Risks and mitigations

AI video pricing and model access are volatile. Mitigation: mark the page `last_verified: 2026-05-08`, use official source links, describe access caveats, and avoid exact plan claims unless official pricing pages support them.
